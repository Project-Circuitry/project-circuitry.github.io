# project-circuitry.github.io
Project Circuitry website

Local preview
----------------

To preview the site locally without Python, you can use one of these options:

- Use Node's one-off server (no install if using `npx`):

```powershell
npx http-server -p 8000
# open http://localhost:8000
```

- Or use `npx serve`:

```powershell
npx serve -l 8000
# open http://localhost:8000
```

- Or use the VS Code *Live Server* extension and open the workspace root.

GitHub Pages
------------

You can publish this repository via GitHub Pages (use the `main` branch, root). Push the files and enable Pages in the repository settings.
